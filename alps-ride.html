The Ride

A road cycling traverse from the Dolomites to the French Alps — the Stelvio, the Gavia, Lake Como, the Lautaret — finishing at the foot of Alpe d'Huez for the Tour de France.

The route

Bolzano → Alpe d'Huez

Bolzano / Bozen → train → Merano → Vinschgau cycle path → Prato allo Stelvio → Stelvio (2,758m) → Bormio [rest] → Gavia (2,621m) → Lake Como [rest] → train → Oulx → Montgenèvre → Briançon → Lautaret → foot of Alpe d'Huez

Each stage below has its own map.
